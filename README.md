# HGF Film Simulation

My implementation of ["A Hyperbolic Geometric Flow for Evolving Films and Foams" \[Sadashige Ishida et al. 2017\]](https://sadashigeishida.bitbucket.io/hgf/index.html)

## Current Status
Under development...

## Dependencies
- libigl
- Eigen
- OpenGL
- glut
- GLFW
- LAPACK

## Compile
    
    mkdir build
    cd build
    cmake ..
    make

## Run

From within the `build` directory just issue:

    ./main
